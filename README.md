"The Band" er en tutorial som tager dig igennem processen med at lave et website. 

Du kan åbne turorialen her.

Hvis du har prøvet at kode i html og css før, kan du vælge "kod selv" udgaven, som du kan åbne her.

Uanset om du vælger den ene eller den anden udgave, må du gerne downloade wordfilen til din computer, så I ikke alle sammen sidder med den samme word-fil.

Teksterne som skal bruges på webløsningen kan du downloade her

Billeder og logoer til løsningen kan du downloade her.
Linket er en zip-fil som indholder billederne til opgaven.
Det fremgår af tutorialen hvordan filerne skal placeres i projektet.

Filerne ovenfor kan alle downloades til din egen computer ved at højreklikke på linket og vælg "save link as..."

Er kodet selv.
